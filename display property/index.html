Display property

What is the display?

This is one of the property, briefly saying, it is to decide the element display format
There are block, inline, inline-block and none.

What is Block?

Block elements break the flow of a document. Width, height, margin and padding are respected p,div,ul, and h1 to h6 are default settings by block.

3. Margin and padding also can be specified

4. Text-align and vertical-align cannot be specified

Only text in the element is centered
If you want to make block element centered, you can add margin right and left
margin:auto auto;

What is the inline

Width and height are ignored. Margin and padding puch elements away HORIZONTALLY but not VERTICLLY a,span and image tag are default settings by inline.



1. Mainly used as part of the text

For example, it is used for elements such as text decorations and links.



2. Lined up side by side with no line breaks between elements

As with the text, when you reach the right end, a line break will occur.

This is a tag and span tag.

What is the inline-block

Behave like an inline element except width, height, margin and padding are respected
